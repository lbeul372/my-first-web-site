HTML이란 무엇인가?

Hypertext Markup Launguage (HTML) is the standdard markup language for creating web pages and web applications. Web browsers receive HTML documents from a web server of from local storage and render them into multimedia web pagers. HTML describes the structure of a web page semantically and originally included cues for the appearance of the document

HTML elemenrs are th building blocks of HTML pages. With HTML constructs, images and other objects, such as interactive forms, may be embedded into the rendered page. It provides a means to creat structured documents by denoting structural semantics for text such as headings, paragraphs, list, links, quotes and other items. HTML elements are delineated by tags, written using angle brackets.
